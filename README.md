#### **Clone the repository**

```bash
git clone git@github.com:Wattarvipavag/vteam14-app.git
```

#### **Change directory**

```bash
cd vteam14-app
```

#### **Install dependencies**

```bash
npm install
```

---

## **Docker environment**

#### **Run the application**

```bash
docker compose up -d
```

or

```bash
docker compose up
```

#### **Stop the application**

```bash
docker compose down
```

---

## **Local environment**

#### **Run the application**

```bash
npm run dev
```

## **Tests**

#### **Run Tests with Vitest**

```bash
npm run test
```
